# 大學生作業 Deadline 管理系統

這是一個使用 **Python Flask + SQLite** 開發的簡單 Web Application，幫助大學生管理作業、報告、考試的截止日期，避免忘記 Deadline。

## 功能簡介

- 新增 / 編輯 / 刪除作業（CRUD）
- 每筆作業欄位：
  - 作業名稱（必填）
  - 課程名稱
  - 截止日期（date）
  - 優先程度（高 / 中 / 低）
  - 備註
  - 是否完成
- 作業清單：
  - 可依截止日期、優先程度、建立時間排序
  - 可依課程名稱篩選
  - 可選擇隱藏已完成的作業
- 提醒效果：
  - 逾期顯示「已過期」
  - 3 天內截止：紅色標示
  - 4～7 天內截止：橘色標示
